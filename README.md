# PresAndo Clothes Shop

Уеб приложение за онлайн магазин за дрехи, създадено като проект по Интернет програмиране.

Сайтът позволява на потребителите да разглеждат продукти, да се регистрират, да влизат в профила си и да добавят продукти в количката.


## Описание на проекта

PresAndo Clothes Shop е уеб сайт за онлайн пазаруване на дрехи.  
Проектът е създаден с цел да демонстрира създаването на уеб приложение с база данни, потребителски акаунти и количка за пазаруване.

Сайтът има модерен дизайн и удобна навигация, която позволява на потребителя лесно да разглежда продуктите.


## Основни функционалности

- Разглеждане на продукти  
- Добавяне на продукти в количката  
- Преглед на количката  
- Регистрация на потребители  
- Вход в профил  
- Потребителски профил  
- Навигация между страниците  
- Начална страница с категории  
- Показване на брой продукти в количката  


## Използвани технологии

- HTML  
- CSS  
- JavaScript  
- C#  
- ASP.NET Core  
- SQL Server  


## Структура на сайта

Сайтът съдържа следните основни страници:

- Начална страница  
- Продукти  
- Количка  
- Профил  
- Вход  
- Регистрация  

### Началната страница включва:

- Нови модели
- Любими модели
- Намаления
- Категории


## База данни

Проектът използва база данни за съхранение на информация за:

- Потребители  
- Продукти  
- Количка  

Базата данни позволява съхранение и извличане на информация за продуктите и потребителите.
