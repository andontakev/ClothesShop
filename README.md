# 🛍️ PresAndo Clothes Shop  
Модерен онлайн магазин, изграден с **ASP.NET Core MVC**.  
Проектът включва начална страница, страница за поверителност, база данни, продукти, количка и тестове.

---

## Функционалности

### Публична част
- Начална страница с модерна визия  
- Категории: Жени, Мъже, Аксесоари  
- Страница за поверителност  
- Продукти (списък + детайли)  
- Количка (добавяне/премахване на продукти)

### Административна част
- Добавяне/редактиране/изтриване на продукти  
- Управление на категории  
- Управление на поръчки  

### Потребители
- Регистрация  
- Вход  
- Изход  

---

## Технологии

Проектът използва:

- **ASP.NET Core MVC**
- **Entity Framework Core**
- **SQL Server / LocalDB**
- **Bootstrap / Custom CSS**
- **xUnit / NUnit** (за тестове)
- **GitHub** за контрол на версиите
